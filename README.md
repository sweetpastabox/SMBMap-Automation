# installation

```
sudo git clone <this repo>
cd SMBMap-Automation
```

## requirements
install requirements by: 

``` 
python3 -m pip install -r requirements.txt
```
## You also need to install : 

SMBMAP (Native on Kali-Linux / Black Arch / ParrotOS)
